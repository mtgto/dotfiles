dotfiles

mtgto's dotfiles.

# Requirements
## Homeshick
First, you install [homeshick](https://github.com/andsens/homeshick) if you didn't.

```bash
brew install homeshick
```

Ahd then, clone this castle.

```
homeshick clone mtgto/dotfiles
```

# Installation
## Powerline
```sh
pip3 install psutil
pip3 install powerline-status
```


# Configuration
## git config
